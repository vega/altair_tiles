Providers
---------
Preconfigured tile providers are provided by the `xyzservices <https://xyzservices.readthedocs.io/en/stable/>`_ package.
The ``xyzservices.providers`` module is accessible as ``altair_tiles.providers``. See `A full look into providers objects - xyzservices <https://xyzservices.readthedocs.io/en/stable/introduction.html>`_ for the documentation.

You can also access ``xyzservices.TileProvider`` at ``altair_tiles.TileProvider`` to configure a custom tile provider. See `API reference - xyzservices <https://xyzservices.readthedocs.io/en/stable/api.html#xyzservices.TileProvider>`_ for the documentation.
